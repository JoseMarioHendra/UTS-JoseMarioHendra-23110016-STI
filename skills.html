Skills

  • HTML

    • Elements

      • Essential components of web documents, including elements like <html>,<head>,<title>, and <body>.

    • Tags

      • Used to delineate the start and end of HTML elements, such as <p> for paragraph or <img>` for image.

    • Attributes

      • Provide extra information about elements, like src for image location or href for link URLs.

    • Hierarchial Structure

      • Allows nesting of elements, like paragraphs within sections or headings within headers.

    • Images

      • Facilitates image inclusion using the <img> element

    • Lists

      • Offers functionality for creating ordered lists <ol> and unordered lists <li>.

    • Forms

      • Allows user data input via forms, including input fields, textareas, and buttons.

    • Semantics

      • Incorporates semantic elements like <header>, <footer>, <footer>, and <footer> for content meaning and structure.

  • CSS

    • Selectors

      • Used to choose HTML elements to style. For example, element selectors ('p', 'div', 'h1', etc.), class selectors (.class').

    • Properties

      • Define the style or appearance of selected elements. For instance, properties like `color` for text color, `font-size` for font size, and `background-color` for background color.

    • Values

      • Values assigned to properties to determine the appearance of elements. Examples include `red`, `20px`, or `#FFFFFF` (hex color code).

    • Box Model

      • Describes how each HTML element is rendered as a box, including margin, border, padding, and content. This affects the layout and size of elements on the page.

    • Positioning

      • Controls the placement of elements on the page, such as `static`, `relative`, `absolute`, or `fixed`. This allows elements to be positioned relative to other elements or the page layout.

    • Colors

      • Specify the text, background, and other element colors on the page. This can be done using color names, hex color codes, or rgba values.
